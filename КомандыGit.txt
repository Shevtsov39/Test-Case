Вступление

Robot Framework - это основанная на Python расширяемая среда автоматизации на основе ключевых слов для приемочного тестирования, разработки, управляемой приемочными тестами (ATDD), разработки, управляемой поведением (BDD), и автоматизации роботизированных процессов (RPA). 
Его можно использовать в распределенных гетерогенных средах, где автоматизация требует использования различных технологий и интерфейсов.
Фреймворк имеет богатую экосистему, состоящую из различных общих библиотек и инструментов, которые разрабатываются как отдельные проекты. 
Для получения дополнительной информации о Robot Framework и экосистеме посетите http://robotframework.org.
